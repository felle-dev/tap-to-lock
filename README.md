**[Tap to Lock](https://github.com/felle-dev/tap-to-lock)** is a simple, lightweight Android app that lets you lock your screen with just a single tap. No frills, no delays—just instant security.

[![Get it on GitHub](https://img.shields.io/badge/Get%20it%20on-GitHub-181717?style=for-the-badge&logo=github&logoColor=white)](https://github.com/felle-dev/tap-to-lock/releases)

## **Features**

- **One-Tap Lock** – Lock your screen instantly with a single tap
- **Lightweight & Fast** – Minimal resource usage for maximum efficiency

## **Tech Stack**

- **Language:** Java
- **Platform:** Android
- **Minimum SDK:** API 24 (Android 7.0)
- **Target SDK:** API 36 (Android 16)

## **Getting Started**

### **Prerequisites**
- [Android Studio](https://developer.android.com/studio) (latest stable version)
- Android device or emulator (API 24+)

### **Installation**
1. Clone the repository:
   ```bash
   git clone https://github.com/felle-dev/tap-to-lock.git
   cd tap-to-lock
   ```
2. Open the project in **Android Studio**.
3. Build and run the app.
