# Top to Lock
A simple Android app to lock your screen with a single tap.

## Features
- Tap to lock the screen instantly
- Lightweight and fast

## Tech Stack
- **Language:** Java
- **Platform:** Android
- **Minimum SDK:** API 24 (Android 7)
- **Target SDK:** API 36 (Android 16)

## Requirements
- Android Studio
- Android SDK
